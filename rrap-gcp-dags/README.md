# Rewrite Dags

## Description

Repository to separate the rewrite dags (derived variables up to reporting) for RRAP that are currently being worked on for RESL and in the future for Re-write.

### State

These dags will be deployed to a separate Airflow instance on the same set of GCVE boxes but under a different DNS name. Eventually, these may be migrated up to the GDAP Composer instance to enable staging runs in GDAP.
